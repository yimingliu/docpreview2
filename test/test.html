The BasicPlugin defines two methods in native code, and provides a scriptable object via NPAPI. It handles the MIMEType application/x-vnd-thallos-test.

  • foo() should return 42.
  • sayHello() will ask the plugin to create a string called "Hello world!" and send it to a callback provided as the first argument -- in this case, handleDataFromPlugin, which prints the text as an alert.
